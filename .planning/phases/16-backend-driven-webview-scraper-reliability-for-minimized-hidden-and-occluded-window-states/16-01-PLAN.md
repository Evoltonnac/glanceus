# Phase 1 Plan 01: Backend Durable Scraper Task Pipeline

## Objective
- **What:** Add backend-owned durable scraper task queue and internal lifecycle APIs.
- **Why:** Remove frontend route/visibility coupling from scraper orchestration.
- **Output:** Executor-integrated task store + claim/heartbeat/complete/fail backend APIs.
