## Problem

用户希望能有意地快速区分其关注的某些卡片，目前 Bento 卡片的视觉风格比较统一，缺乏一种简单、非侵入性的方式来标注或突出显示特定的卡片。

## Solution

在不影响现有 UI 整体风格的情况下，为 Bento 卡片增加自定义颜色或其他视觉区分的功能：

1. **自定义基准色**：允许用户通过配置文件或 UI 为特定的卡片设置一种基准颜色（如 Border 颜色、Icon 背景色或卡片顶部的细线）。
2. **渐变装饰**：支持在卡片背景或边缘应用微妙的渐变，使其与其他卡片有所区别。
3. **视觉状态增强**：增加一个“关注”或“标注”状态，通过不同的图标或背景纹理来区分。
4. **UI 配置入口**：在卡片的配置面板中增加颜色选择或预设风格选择。
